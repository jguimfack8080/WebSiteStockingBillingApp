Produkte und ihre Barcodes verwalten

Admin

Produkte anlegen, bearbeiten und finden; einen Barcode für das Scannen zuordnen.

Präsentation

Produkte sind die Basis von Bestand und Verkauf. Jedes Produkt hat einen Preis, eine Menge und kann einen Barcode tragen, der es sofort scannbar macht.

Zugang

Aus der Inventaransicht ("Bestand verwalten") oder dem Produkte-Bildschirm. Das Scannen eines unbekannten Codes öffnet dort direkt die Anlage mit vorausgefülltem Code.

Oberfläche

  • Suche und Filter (niedrig / leer): Ein Produkt finden oder Warnungen anvisieren.
  • Produkt anlegen / bearbeiten: Name, Preis, Menge, Kategorie, Barcode, Schwellen, Ablauf.
  • Scan (Handscanner / Kamera): Bekannter Scan: öffnet den Datensatz. Unbekannter Scan: öffnet die Anlage mit vorausgefülltem Code.

Schritt-für-Schritt-Anleitung

  1. Aktion: Öffnen Sie den Produkte-Bildschirm. Zum Anlegen "+" tippen (oder unbekannten Code scannen). Ergebnis: Das Produktformular öffnet sich (Code vorausgefüllt bei Scan).
  2. Aktion: Name, Preis, Menge, Kategorie und möglichst den Barcode eingeben; speichern. Ergebnis: Das Produkt wird angelegt und ist scannbar, wenn es einen Barcode hat. Prüfung: Scannen Sie das Produkt zur Prüfung der Erkennung.

Beispiele

[einfach] Ein Produkt durch Scannen anlegen: Im Produkte-Bildschirm scannen Sie einen neuen Artikel: das Formular öffnet sich mit vorausgefülltem Code, Sie geben Name und Preis ein und bestätigen.

Sonderfälle

Ein ohne Barcode angelegtes Produkt ist erst scannbar, wenn Sie einen zuordnen.

Tipps

  • Legen Sie Warnschwellen (niedriger Bestand) bei der Anlage fest: das Dashboard warnt automatisch.

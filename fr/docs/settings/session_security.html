Sécurité de session et déconnexion automatique

Tous les rôles

Comprendre pourquoi l'application peut vous déconnecter (session expirée, compte désactivé, inactivité), comment rester connecté, et régler le délai d'inactivité.

Présentation

Pour protéger vos données, l'application peut mettre fin à votre session dans trois cas : (1) votre session a expiré (après une longue durée), (2) un administrateur a désactivé votre compte, (3) vous êtes resté INACTIF trop longtemps (aucune action). Dans tous les cas, vous êtes ramené à l'écran de connexion avec un message qui explique pourquoi.

La déconnexion pour inactivité est importante sur un poste partagé (une caisse) : si vous vous éloignez sans vous déconnecter, personne ne peut agir à votre place. Avant de vous déconnecter, l'application vous prévient avec un compte à rebours et vous laisse le choix de rester connecté.

Accès

Le réglage du délai d'inactivité se trouve dans Réglages > Sécurité > "Déconnexion automatique". L'avertissement d'inactivité apparaît tout seul quand vous restez sans rien faire. Le message expliquant une déconnexion s'affiche sur l'écran de connexion.

Interface

  • Avertissement d'inactivité : une fenêtre "Toujours là ?" avec un compte à rebours. Deux boutons : "Rester connecté" (poursuit la session) et "Se déconnecter". Si le compte à rebours atteint zéro, vous êtes déconnecté automatiquement.
  • Message de raison (écran de connexion) : une bannière sur l'écran de connexion indique pourquoi la session a pris fin : session expirée, compte désactivé, ou inactivité.
  • Sélecteur "Déconnexion automatique" : dans Réglages > Sécurité : un menu pour choisir le délai d'inactivité (5, 10, 15, 30 ou 60 minutes), ou "Jamais" pour le désactiver. Valeur par défaut : 10 minutes.

Procédure pas à pas

  1. Pour changer le délai : ouvrez Réglages > Sécurité > "Déconnexion automatique" et choisissez une durée (ou "Jamais"). -> Le nouveau délai est appliqué immédiatement et conservé sur cet appareil (même après redémarrage). (Vérif : rouvrez le menu : la valeur choisie est bien affichée.)
  2. Quand l'avertissement "Toujours là ?" apparaît : cliquez sur "Rester connecté" pour poursuivre, ou "Se déconnecter" pour fermer la session tout de suite. -> "Rester connecté" remet le compteur à zéro. Sinon, à la fin du compte à rebours, vous êtes ramené à la connexion.

Exemples

Je reviens à temps : vous laissez la caisse 14 minutes. L'avertissement apparaît : vous cliquez "Rester connecté". Vous continuez sans avoir à ressaisir votre mot de passe.

Pause déjeuner : vous partez 30 minutes sans vous déconnecter (délai réglé à 15 min). À votre retour, vous êtes sur l'écran de connexion avec le message "déconnecté pour inactivité". Vous vous reconnectez : tout est intact.

Compte désactivé en cours de session : un administrateur désactive votre compte pendant que vous travaillez. À votre action suivante, vous êtes immédiatement déconnecté avec le message "compte désactivé". C'est normal : contactez votre administrateur.

Option "Jamais" : la déconnexion par inactivité est désactivée. À réserver à un appareil PERSONNEL de confiance ; déconseillé sur une caisse partagée.

Après une mise à jour de sécurité, il peut être nécessaire de se reconnecter une fois, même si rien d'autre n'a changé. C'est normal.

Au moment de payer, un double-clic ou un réessai après coupure réseau n'encaisse jamais deux fois : l'opération est protégée.

Messages d'erreur

Votre session a expiré. Veuillez vous reconnecter.
Cause : la durée de validité de votre connexion a été atteinte, ou la connexion n'est plus valide. - Solution : reconnectez-vous avec votre e-mail et votre mot de passe.
Votre compte a été désactivé. Contactez votre administrateur.
Cause : un administrateur a désactivé votre compte ; vos accès sont suspendus. - Solution : contactez votre administrateur pour réactiver le compte.

Conseils

  • Sur une caisse partagée, choisissez un délai court (5 ou 10 min) et déconnectez-vous toujours en partant.
  • Ne restez connecté que sur un appareil de confiance : tant que la session est ouverte, une autre personne pourrait agir en votre nom.

Consulter l'historique des ventes

AdminManagerCaissier

Retrouver une vente, voir son détail, l'annuler ou la rembourser, exporter la liste.

Présentation

L'historique liste les ventes réalisées, avec leur montant, leur client et leur statut. Il sert à vérifier, retrouver une facture, corriger une erreur (annulation) ou rembourser.

Accès

Touchez l'icône Historique dans la barre du haut (caissier) ou la carte "Historique des ventes" du tableau de bord (manager / admin).

Description de l'interface

  • Filtres de statut (toutes / terminées / annulées) : restreignent la liste affichée.
  • Filtrer par vendeur (admin / manager) : pour l'admin et le manager uniquement : un menu déroulant "Filtrer par vendeur" permet de n'afficher que les ventes d'un caissier précis (ou "Tous les vendeurs"). Le caissier, lui, ne voit que ses propres ventes - ce menu n'apparaît pas pour lui, et le serveur garantit qu'il ne peut pas voir celles des autres.
  • Détail d'une vente : affiche les articles, le client, le mode de paiement, la monnaie rendue, la date.
  • Annuler / Rembourser : permet d'annuler une vente (avec motif) ou de la rembourser.
  • Exporter : exporte la liste (selon les droits) pour archivage ou analyse.

Procédure pas à pas

  1. Ouvrez l'historique et, si besoin, filtrez par statut. -> La liste se met à jour.
  2. Touchez une vente pour voir son détail. -> Le détail complet s'affiche.
  3. Pour corriger : utilisez Annuler (saisissez un motif) ou Rembourser, puis confirmez. -> Le statut de la vente est mis à jour en conséquence. (Vérif : vérifiez que le statut est passé à annulée / remboursée.)

Exemples

[Simple] Retrouver une vente du jour : vous ouvrez l'historique, filtrez sur "terminées" et touchez la vente concernée pour revoir son ticket.

[Intermédiaire] Annuler une erreur de caisse : une vente a été enregistrée deux fois : vous ouvrez le doublon, choisissez Annuler, saisissez le motif et confirmez.

[Avancé] Voir les ventes d'un caissier précis (admin / manager) : vous ouvrez l'historique, déroulez "Filtrer par vendeur" et choisissez un caissier : la liste n'affiche plus que ses ventes. Pour une analyse comparée complète (CA, panier moyen, classement), utilisez plutôt l'écran "Performance des caissiers".

Cas particuliers

Les possibilités d'annulation / remboursement / export peuvent dépendre de votre rôle et des règles de l'entreprise.

Conseils

  • Renseignez toujours un motif clair lors d'une annulation : il reste tracé pour le suivi.

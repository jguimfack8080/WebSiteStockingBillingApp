Connexion biométrique (empreinte / visage)

Tous les rôles

Se connecter sans retaper son mot de passe, avec l'empreinte, le visage ou Windows Hello.

Présentation

La connexion biométrique vous évite de saisir votre mot de passe à chaque fois : l'appareil vérifie votre empreinte ou votre visage, puis ouvre votre session.

Vos identifiants sont conservés uniquement sur CET appareil, dans son coffre-fort sécurisé, et ne sont utilisés qu'après une vérification biométrique réussie. Ils ne sont jamais envoyés ailleurs.

Cette option n'apparaît que si votre appareil dispose de la biométrie (empreinte, reconnaissance faciale ou Windows Hello). Sur le web, elle n'est pas disponible : on utilise le mot de passe.

Accès

Sur l'écran de connexion : la case "Activer la connexion biométrique" (sous le mot de passe) et, une fois activée, le bouton "Se connecter avec la biométrie".

Dans Paramètres > Sécurité > "Connexion biométrique" : pour l'activer (avec confirmation du mot de passe) ou la désactiver.

Interface

  • Case "Activer la connexion biométrique" (connexion) : cochez-la avant de vous connecter pour mémoriser vos identifiants sur cet appareil.
  • Bouton "Se connecter avec la biométrie" : lance la vérification (empreinte/visage) puis ouvre votre session sans mot de passe.
  • Interrupteur "Connexion biométrique" (Paramètres > Sécurité) : active (après confirmation du mot de passe) ou désactive la biométrie. La désactivation efface les identifiants mémorisés.

Procédure pas à pas

  1. À la connexion, saisissez e-mail et mot de passe, cochez "Activer la connexion biométrique", puis validez. -> Votre session s'ouvre et vos identifiants sont mémorisés en sécurité sur l'appareil.
  2. La prochaine fois, ouvrez simplement l'application : l'invite biométrique apparaît automatiquement (un bouton "Se connecter avec la biométrie" reste disponible si vous avez annulé l'invite). -> Dès que votre visage ou votre empreinte est reconnu, vous entrez directement, sans rien toucher ni confirmer. (Vérif : vous arrivez sur votre tableau de bord sans avoir saisi de mot de passe.)
  3. Vous pouvez aussi l'activer plus tard via Paramètres > Sécurité > "Connexion biométrique" (mot de passe demandé pour confirmer). -> Une fois confirmé, le bouton biométrique apparaîtra à la prochaine connexion.

Exemples

Activer à la première connexion : le caissier se connecte, coche la case biométrie, valide. Le lendemain, il touche le bouton, pose son doigt : la caisse s'ouvre en une seconde.

Activer depuis les Paramètres : déjà connecté, le manager va dans Paramètres > Sécurité, active "Connexion biométrique", retape son mot de passe pour confirmer : c'est prêt pour la prochaine fois.

Sur le web (navigateur), la connexion biométrique n'est pas disponible : utilisez votre mot de passe.

Si votre mot de passe est modifié, la connexion biométrique se désactive automatiquement : reconnectez-vous avec le nouveau mot de passe, puis réactivez-la.

Messages d'erreur

Mot de passe incorrect (activation)
Cause : le mot de passe saisi pour confirmer l'activation ne correspond pas à votre compte. - Solution : ressaisissez votre mot de passe actuel ; en cas de doute, utilisez "Mot de passe oublié".
Authentification biométrique impossible
Cause : la biométrie a échoué ou les identifiants mémorisés ne sont plus valides. - Solution : connectez-vous avec votre mot de passe, puis réactivez la connexion biométrique.

Conseils

  • N'activez la biométrie que sur VOTRE appareil personnel. Sur un appareil partagé, laissez-la désactivée.
  • Pour retirer l'accès sur un appareil, désactivez "Connexion biométrique" dans Paramètres > Sécurité : les identifiants mémorisés sont effacés.

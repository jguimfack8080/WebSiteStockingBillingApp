Réinitialiser un mot de passe oublié

Tous les rôles

Recevoir un code par e-mail et définir un nouveau mot de passe, en 3 étapes.

Présentation

Si vous avez oublié votre mot de passe, vous pouvez le réinitialiser vous-même, sans attendre l'administrateur, grâce à un code envoyé sur votre e-mail.

Accès

Sur l'écran de connexion, touchez le lien "Mot de passe oublié" sous le champ mot de passe.

Procédure pas à pas

  1. Étape 1 : saisissez votre e-mail et votre date de naissance, puis touchez "Envoyer le code". -> Un code est envoyé à votre adresse e-mail. (Vérif : consultez votre boîte mail (et les courriers indésirables).)
  2. Étape 2 : saisissez le code reçu, puis touchez "Vérifier". -> Le code est validé et vous passez à l'étape suivante.
  3. Étape 3 : saisissez votre nouveau mot de passe, puis touchez "Réinitialiser". -> Le mot de passe est changé ; vous pouvez vous connecter avec le nouveau.

Exemples

Le code a une durée de validité limitée. S'il a expiré, recommencez l'étape 1 pour en recevoir un nouveau.

Messages d'erreur

Code invalide ou expiré
Cause : le code saisi est erroné ou n'est plus valable. - Solution : recommencez l'étape 1 pour recevoir un nouveau code, puis ressaisissez-le rapidement.

Conseils

  • Copiez-collez le code depuis l'e-mail pour éviter toute faute de frappe.

Scanner un code-barres (douchette, caméra ou saisie)

Tous les rôles

Comprendre les 3 façons de scanner et le scan automatique dès qu'on est sur un écran de flux.

Présentation

Le scan permet de retrouver un produit instantanément par son code-barres, dans la vente, l'inventaire et la recherche de produit. C'est beaucoup plus rapide et fiable que la saisie à la main.

Comportement professionnel : dès que vous êtes sur un écran de flux (Vente, Produits, Mouvements), un scan à la douchette est capté et traité AUTOMATIQUEMENT, sans appuyer sur aucun bouton.

Accès

Le scan automatique (douchette) est actif sur les écrans Vente, Produits et Mouvements. Un bouton avec une icône de scan y est aussi présent pour la caméra (mobile) et la saisie manuelle du code.

Interface

  • Douchette (lecteur USB / Bluetooth) : fonctionne sur tous les appareils. Aucun bouton à presser : on scanne, l'action se déclenche.
  • Bouton scan (caméra) : ouvre la caméra pour viser un code (Android et navigateur web). Sur un poste Windows sans caméra, utilisez une douchette connectée ou la saisie manuelle.
  • Saisie manuelle : permet de taper le code à la main, disponible sur tous les appareils.

Procédure pas à pas

  1. Ouvrez l'écran voulu (Vente, Produits ou Mouvements). -> Le scan à la douchette est immédiatement actif (rien à activer).
  2. Scannez l'article avec la douchette. -> Le produit est retrouvé et l'action de l'écran se déclenche (ajout au panier, fiche, mouvement). Un retour sonore confirme. (Vérif : vérifiez que le bon produit a été pris en compte.)
  3. Sans douchette : touchez le bouton scan pour la caméra, ou tapez le code à la main. -> Le même traitement s'applique.

Exemples

Caisse avec douchette : sur l'écran de vente, vous scannez 5 articles à la suite : chacun s'ajoute au panier tout seul.

Téléphone sans douchette : sur mobile, vous touchez le bouton scan et visez le code-barres avec la caméra.

Pendant que vous tapez dans un champ (recherche, quantité), le scan automatique ne vous dérange pas : il ne prend la main que si aucun champ n'est en cours de saisie.

Sur les écrans avec une barre de recherche (Produits, Mouvements), vous pouvez scanner même si le curseur est dans la recherche : un vrai code-barres est reconnu comme un scan et traité tout de suite, tandis que votre frappe au clavier reste une recherche normale. Vous n'avez donc jamais besoin de cliquer ailleurs avant de scanner.

Sur Windows (poste de bureau), la caméra n'est pas disponible : utilisez une douchette ou la saisie manuelle.

Messages d'erreur

Code inconnu
Cause : aucun produit de votre entreprise n'est associé à ce code-barres. - Solution : sur l'écran Produits, un code inconnu ouvre directement la création du produit avec le code pré-rempli. Ailleurs, créez d'abord le produit.

Conseils

  • Une douchette est la solution la plus rapide en caisse : branchez-la, elle se comporte comme un clavier, aucune configuration n'est requise.

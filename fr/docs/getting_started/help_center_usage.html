Utiliser le Centre d'Aide et la recherche

Tous les rôles

Trouver rapidement une réponse : parcourir par catégorie ou rechercher par mot-clé.

Présentation

Le Centre d'Aide explique, pas à pas, comment utiliser l'application. Il n'affiche que les fonctions accessibles à votre rôle, pour éviter toute confusion.

Accès

Touchez l'icône d'aide (point d'interrogation) dans la barre du haut de votre tableau de bord.

Interface

  • Barre de recherche : tapez un mot (ex : "vente", "facture") : les fiches correspondantes apparaissent immédiatement.
  • Liste des catégories : regroupe les fiches par thème (Ventes, Stock, etc.).

Procédure pas à pas

  1. Ouvrez le Centre d'Aide via l'icône d'aide. -> La liste des catégories et la barre de recherche s'affichent.
  2. Tapez un mot-clé, ou touchez une catégorie puis une fiche. -> La fiche s'ouvre avec ses sections (présentation, accès, étapes, exemples, erreurs, conseils).

Conseils

  • Utilisez la recherche plutôt que de parcourir : un seul mot suffit le plus souvent à trouver la bonne fiche.

Consulter les avoirs

Manager

Accéder à la liste des avoirs (notes de crédit) générés par les remboursements. Lecture seule.

Présentation

Les avoirs sont générés automatiquement à chaque remboursement. Cet écran permet de les consulter et de les filtrer par période. Lecture seule.

Accès

Tableau de bord > section Finance > tuile 'Avoirs'. Permission : sales.refund.

Description de l'interface

  • Liste : liste paginée : numéro, montant, devise, motif, date. Chargement infini au défilement.

Procédure pas à pas

  1. Tableau de bord > 'Avoirs'. -> Liste des avoirs affichée.

Erreur de chargement
Cause : connexion perdue. Solution : appuyez sur Réessayer.

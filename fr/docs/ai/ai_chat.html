Utiliser l'Assistant IA

Tous les rôles

Poser des questions métier en français (ou anglais / allemand) et recevoir des réponses basées sur les données réelles de votre entreprise.

Présentation

L'Assistant IA vous permet de poser des questions sur votre activité en langage naturel, sans avoir à naviguer dans les menus ou construire des rapports manuellement. Il répond uniquement à partir des données réelles de votre entreprise (produits, ventes, clients, mouvements de stock) : il ne peut pas accéder aux données d'une autre entreprise.

Chaque réponse peut inclure la liste des sources utilisées (table de données concernée), vous permettant de vérifier d'où vient l'information.

Accès

Depuis le tableau de bord, faites défiler jusqu'à la section "Intelligence Artificielle", puis touchez la carte "Assistant IA".

Interface

  • Zone de messages : affiche la conversation en cours. Les bulles bleues (droite) sont vos questions ; les bulles grises (gauche) sont les réponses de l'assistant.
  • Chips de suggestion : affichés quand la conversation est vide : proposent des questions types à toucher directement pour démarrer.
  • Champ de saisie : en bas de l'écran. Écrivez votre question puis touchez la flèche d'envoi (ou appuyez sur Entrée).
  • Indicateur "réflexion" : trois points animés apparaissent pendant que l'assistant prépare sa réponse. L'envoi est désactivé pendant ce temps.
  • Chips de sources : affichés sous chaque réponse de l'assistant. Indiquent les tables de données consultées (ex. : Produits, Ventes, Clients).
  • Bouton "Effacer" : en haut à droite, disponible quand la conversation contient des messages. Permet de repartir de zéro.

Procédure pas à pas

  1. Ouvrez l'Assistant IA depuis le tableau de bord (section "Intelligence Artificielle"). -> L'écran de chat s'ouvre avec des suggestions de questions.
  2. Tapez votre question dans le champ en bas, ou touchez une suggestion. -> Votre message apparaît en bleu à droite. L'indicateur de réflexion s'affiche.
  3. Attendez quelques secondes (2-5 secondes en général) que l'assistant prépare sa réponse. -> La réponse apparaît en gris à gauche, avec les sources consultées en dessous. (Vérif : vérifiez les sources affichées pour savoir quelles données ont été utilisées.)
  4. Posez une question de suivi ou continuez la conversation librement. -> Chaque question est traitée indépendamment : l'assistant n'a pas de mémoire des échanges précédents.

Exemples

Ruptures de stock : Question : "Quels produits sont en rupture de stock ?" - L'assistant liste les produits dont le stock est à zéro ou en dessous du seuil minimal, avec leur catégorie.

Chiffre d'affaires : Question : "Quel est mon chiffre d'affaires ce mois ?" - L'assistant calcule le total des ventes du mois en cours à partir des données réelles.

Analyse clients : Question : "Quels sont mes 5 meilleurs clients par chiffre d'affaires ?" - L'assistant identifie et classe les clients les plus importants avec leur volume d'achat.

L'assistant ne se souvient pas des questions précédentes : chaque question est traitée indépendamment. Reformulez avec le contexte complet si nécessaire.

Pour des données très récentes (il y a moins d'une minute), l'assistant peut ne pas encore les avoir indexées. Les données sont mises à jour en temps quasi réel.

Si l'assistant ne trouve pas de données pertinentes, il indique qu'il n'a pas les informations nécessaires. Reformulez votre question ou vérifiez que des données existent dans le module concerné.

Messages d'erreur

L'assistant est temporairement indisponible.
Cause : le service IA est momentanément surchargé ou en maintenance. - Solution : réessayez dans 1 à 2 minutes. Si le problème persiste, contactez votre administrateur.
Trop de questions envoyées. Attendez une minute.
Cause : le service limite à 30 questions par minute pour garantir la qualité du service. - Solution : attendez 60 secondes avant d'envoyer une nouvelle question.
L'assistant IA n'est pas encore configuré pour votre entreprise.
Cause : l'indexation initiale des données n'a pas encore été effectuée par l'administrateur système. - Solution : contactez votre administrateur pour qu'il active l'assistant IA pour votre entreprise.
Votre message ne peut pas être traité. Reformulez votre question.
Cause : le message contenait des éléments non autorisés (instructions système, tentatives d'accès à d'autres entreprises). - Solution : posez une question métier simple et directe sur votre activité (ventes, stock, clients).

Conseils

  • Soyez précis : "Combien d'unités de yaourt fraise ai-je en stock ?" donnera une meilleure réponse que "Combien de produits j'ai ?"
  • Vous pouvez mentionner des périodes : "ce mois", "la semaine dernière", "en janvier" pour des questions temporelles.
  • Utilisez les chips de suggestion au premier accès pour découvrir ce que l'assistant sait faire.
  • La conversation n'est pas sauvegardée entre sessions : si vous fermez l'écran, elle disparaît.

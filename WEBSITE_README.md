# Site vitrine BizCoRa

La documentation du site a ete consolidee dans [README.md](README.md).

Resume : site vitrine statique (HTML / CSS / JavaScript natif, sans build ni
dependance) dedie a la presentation et a la vente de l'application BizCoRa.

Pages : 6 pages par langue dans `/fr`, `/en`, `/de` (accueil, fonctionnalites,
tarifs, FAQ, a propos, contact). Voir le [README](README.md) pour la structure,
le design, le lancement en local, le deploiement et la gestion du domaine via `.env`.
